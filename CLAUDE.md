# 数学自学教材（1-9 年级）— 项目约定

## 项目概述

面向中国 1-9 年级学生的中文数学自学教材。内容覆盖中国课标（小学一年级至初中三年级），教学风格借鉴美国教材：探索先于讲解、概念驱动、螺旋式深化、真实情境连接。

## 文件结构

- `00-introduction/` — 致读者、学习路径、符号约定、目录总览
- `01-numbers-and-operations/` — 数与运算（§1.1-§1.8）
- `02-algebra/` — 代数（§2.1-§2.11）
- `03-geometry/` — 几何（§3.1-§3.10）
- `04-statistics-and-probability/` — 统计与概率（§4.1-§4.6）
- `assets/` — SVG 图形（按模块子目录组织）
- `output/` — 构建产物（git ignored）

## 知识点五步结构

每个 §X.Y 知识点按固定顺序编写：

1. **引入情境（Explore）** — 具体问题或生活场景
2. **概念建立（Build Understanding）** — 直觉 → 可视化 → 定义 → "为什么"
3. **典型例题（Worked Examples）** — 分步骤详解，每步附说明
4. **关键总结（Key Takeaways）** — 3-5 句核心要点
5. **练一练（Practice）** — 习题（答案在文件末尾"参考答案"节）

## LaTeX 规范

- 行内公式：` $...$ `；行间公式：`$$...$$`
- 不等号：`\geq` / `\leq`（不用 `\geqslant`）
- 绝对值：`\lvert a \rvert`（不用 `|a|`）
- 整除/条件：`\mid`
- 多行对齐：`aligned` 环境（不用 `align`）
- 中文文本：`\text{...}` 包裹

## SVG 图形规范

- 独立 SVG 文件存于 `assets/<模块>/` 目录
- Markdown 引用：`![描述](../assets/<模块>/filename.svg)`
- 每个 SVG 文件头部加注释：`<!-- 描述：... -->`
- 宽度建议 ≤500px

## Markdown 标题层级

- `#` — 文件标题
- `##` — 大节 / 参考答案
- `###` — §X.Y 知识点
- `####` — 五步子结构（引入情境、概念建立等）

## 交叉引用

- 格式：`§1.5`、`§2.10`
- 每个知识点开头标注前置知识：`> **前置知识**：§A.B, §C.D`

## 年级密度规则

- 1-5 年级内容：精简（基础概念 + 核心例题）
- 6-9 年级内容：详尽（完整五步结构 + 多例题 + 充分练习）

## 参考答案格式

不使用 `<details>` 折叠（PDF/EPUB 不支持）。每个章节文件末尾设独立 `## 参考答案` 节，用 `---` 分隔线与正文区隔。

## 统计范围限制（按中国课标）

- 包含：方差（八年级）、总体与样本
- 离散程度以极差和方差为主
- 严格排除高中内容：标准差、正态分布、条件概率、独立性检验、排列组合公式、相关系数、回归方程、随机变量、期望值

## 构建

宿主机只需 Docker。运行 `make pdf` 或 `make epub` 即可在容器中构建。
