## 分数的意义

### 引入情境（Explore）

妈妈把一个苹果平均切成 $4$ 块，你吃了其中的 $1$ 块。你吃了这个苹果的多少？

### 概念建立（Build Understanding）

把一个物体或一个整体**平均分**成若干份，取其中的一份或几份，就可以用**分数**来表示。

$$\frac{1}{4}$$

读作"四分之一"。其中 $4$ 叫做**分母**，表示平均分的总份数； $1$ 叫做**分子**，表示取的份数。中间的横线叫**分数线**。

更一般地：

$$\frac{a}{b} \quad (b \neq 0)$$

表示把整体平均分成 $b$ 份，取其中 $a$ 份。

分数也可以理解为**除法的结果**： $a \div b = \frac{a}{b}$ 。

### 典型例题（Worked Examples）

**例 1.** 一条线段平均分成 $5$ 段，涂色了其中的 $3$ 段，涂色部分用分数怎么表示？

**解：** 平均分成 $5$ 份，取 $3$ 份，即 $\frac{3}{5}$ 。

**例 2.** 把 $3$ 块饼平均分给 $4$ 个小朋友，每人分到多少？

**解：** $3 \div 4 = \frac{3}{4}$ ，每人分到 $\frac{3}{4}$ 块饼。

### 关键总结（Key Takeaways）

- 分数表示"平均分之后取几份"。
- 分母表示总份数，分子表示取的份数，分母不能为 $0$ 。
- 分数与除法的关系： $a \div b = \frac{a}{b}$ 。

### 练一练（Practice）

1. 把一块蛋糕平均分成 $8$ 份，吃了 $3$ 份，吃了几分之几？
2. $5 \div 6$ 用分数表示是多少？
3. $\frac{7}{10}$ 表示把整体平均分成几份，取了几份？

## 真分数、假分数与带分数

### 引入情境（Explore）

小明吃了 $\frac{3}{4}$ 个苹果，小红吃了 $\frac{5}{4}$ 个苹果。小红怎么能吃超过一个苹果呢？

### 概念建立（Build Understanding）

- **真分数**：分子小于分母的分数，如 $\frac{3}{4}$ 。真分数小于 $1$ 。
- **假分数**：分子大于或等于分母的分数，如 $\frac{5}{4}$ 、 $\frac{4}{4}$ 。假分数大于或等于 $1$ 。
- **带分数**：由一个整数和一个真分数组成，如 $1\frac{1}{4}$ 。

假分数和带分数可以互化：

$$\frac{5}{4} = 1\frac{1}{4}$$

方法： $5 \div 4 = 1$ 余 $1$ ，所以商 $1$ 为整数部分，余数 $1$ 为分子，分母不变。

反过来： $1\frac{1}{4} = \frac{1 \times 4 + 1}{4} = \frac{5}{4}$ 。

### 典型例题（Worked Examples）

**例 1.** 把 $\frac{11}{3}$ 化成带分数。

**解：** $11 \div 3 = 3$ 余 $2$ ，所以 $\frac{11}{3} = 3\frac{2}{3}$ 。

**例 2.** 把 $2\frac{3}{7}$ 化成假分数。

**解：** $\frac{2 \times 7 + 3}{7} = \frac{17}{7}$ 。

### 关键总结（Key Takeaways）

- 真分数 $< 1$ ，假分数 $\geq 1$ 。
- 假分数化带分数：用分子除以分母。
- 带分数化假分数：整数部分乘分母加分子。

### 练一练（Practice）

1. 从 $\frac{3}{5}$ 、 $\frac{7}{4}$ 、 $\frac{6}{6}$ 、 $\frac{2}{9}$ 中选出真分数和假分数。
2. 把 $\frac{17}{5}$ 化成带分数。
3. 把 $3\frac{2}{5}$ 化成假分数。

## 分数的基本性质

### 引入情境（Explore）

拿两张同样大小的纸条。第一张平均分成 $2$ 份，涂 $1$ 份，得到 $\frac{1}{2}$ 。第二张平均分成 $4$ 份，涂 $2$ 份，得到 $\frac{2}{4}$ 。涂色的面积一样大吗？

### 概念建立（Build Understanding）

**分数的基本性质**：分数的分子和分母同时乘以（或除以）同一个不为零的数，分数的大小不变。

$$\frac{a}{b} = \frac{a \times k}{b \times k} = \frac{a \div m}{b \div m} \quad (k \neq 0,\; m \neq 0)$$

例如： $\frac{1}{2} = \frac{2}{4} = \frac{3}{6} = \frac{50}{100}$ 。

这条性质有两个重要应用：

**约分**：把一个分数的分子和分母同时除以它们的公因数，化成最简分数。

$$\frac{12}{18} = \frac{12 \div 6}{18 \div 6} = \frac{2}{3}$$

**通分**：把异分母分数化成同分母分数，以便比较大小或进行加减运算。

$$\frac{2}{3} \text{ 和 } \frac{3}{5} \longrightarrow \frac{10}{15} \text{ 和 } \frac{9}{15}$$

通分的关键：找两个分母的**最小公倍数**作为公分母。 $\text{最小公倍数}(3, 5) = 15$ 。

### 典型例题（Worked Examples）

**例 1.** 把 $\frac{24}{36}$ 约分。

**解：** $24$ 和 $36$ 的最大公因数是 $12$ ：

$$\frac{24}{36} = \frac{24 \div 12}{36 \div 12} = \frac{2}{3}$$

**例 2.** 比较 $\frac{3}{4}$ 和 $\frac{5}{7}$ 的大小。

**解：** 通分。 $\text{最小公倍数}(4, 7) = 28$ 。

$$\frac{3}{4} = \frac{21}{28}, \quad \frac{5}{7} = \frac{20}{28}$$

因为 $21 > 20$ ，所以 $\frac{3}{4} > \frac{5}{7}$ 。

### 关键总结（Key Takeaways）

- 分子分母同乘或同除以同一个非零数，分数大小不变。
- 约分：除以最大公因数，化为最简分数。
- 通分：找最小公倍数作公分母，再比较或运算。

### 练一练（Practice）

1. 约分： $\frac{15}{45}$ 。
2. 约分： $\frac{28}{42}$ 。
3. 比较 $\frac{5}{6}$ 和 $\frac{7}{9}$ 的大小。
4. 把 $\frac{2}{3}$ 、 $\frac{3}{4}$ 、 $\frac{5}{6}$ 通分。

## 分数的加减法

### 引入情境（Explore）

小明做作业用了 $\frac{1}{3}$ 小时，然后看课外书用了 $\frac{1}{4}$ 小时。他一共用了多长时间？这两个分数的分母不一样，怎么加？

### 概念建立（Build Understanding）

**同分母分数加减法**：分母不变，分子相加或相减。

$$\frac{2}{7} + \frac{3}{7} = \frac{2 + 3}{7} = \frac{5}{7}$$

**异分母分数加减法**：先通分，再按同分母分数加减。

$$
\begin{aligned}
\frac{1}{3} + \frac{1}{4} &= \frac{4}{12} + \frac{3}{12} = \frac{7}{12}
\end{aligned}
$$

**带分数加减法**：整数部分和分数部分分别运算。

$$1\frac{1}{3} + 2\frac{1}{4} = 3 + \frac{4}{12} + \frac{3}{12} = 3\frac{7}{12}$$

### 典型例题（Worked Examples）

**例 1.** 计算 $\frac{5}{6} - \frac{1}{4}$ 。

**解：** 通分。 $\text{最小公倍数}(6, 4) = 12$ 。

$$\frac{5}{6} - \frac{1}{4} = \frac{10}{12} - \frac{3}{12} = \frac{7}{12}$$

**例 2.** 计算 $3\frac{1}{5} + 1\frac{3}{10}$ 。

**解：** 整数部分： $3 + 1 = 4$ 。分数部分通分：

$$\frac{1}{5} + \frac{3}{10} = \frac{2}{10} + \frac{3}{10} = \frac{5}{10} = \frac{1}{2}$$

所以 $3\frac{1}{5} + 1\frac{3}{10} = 4\frac{1}{2}$ 。

**例 3.** 计算 $5\frac{1}{6} - 2\frac{3}{4}$ 。

**解：** 通分。 $\text{最小公倍数}(6, 4) = 12$ 。

$$5\frac{1}{6} - 2\frac{3}{4} = 5\frac{2}{12} - 2\frac{9}{12}$$

分数部分 $\frac{2}{12} < \frac{9}{12}$ ，不够减，从整数部分借 $1$ ：

$$= 4\frac{14}{12} - 2\frac{9}{12} = 2\frac{5}{12}$$

### 关键总结（Key Takeaways）

- 同分母直接加减分子。
- 异分母先通分再加减。
- 带分数减法中分数部分不够减时，从整数部分借 $1$ 。
- 结果要约分为最简分数。

### 练一练（Practice）

1. 计算： $\frac{2}{3} + \frac{5}{6}$ 。
2. 计算： $\frac{7}{8} - \frac{2}{3}$ 。
3. 计算： $4\frac{1}{2} - 1\frac{5}{6}$ 。

## 分数的乘法

### 引入情境（Explore）

一桶油有 $\frac{4}{5}$ 千克，用去了这桶油的 $\frac{1}{2}$ 。用去了多少千克？" $\frac{4}{5}$ 的一半"该怎么算？

### 概念建立（Build Understanding）

**分数乘整数**：分子乘以整数，分母不变。

$$\frac{2}{5} \times 3 = \frac{2 \times 3}{5} = \frac{6}{5} = 1\frac{1}{5}$$

**分数乘分数**：分子相乘作分子，分母相乘作分母。

$$\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$$

例如： $\frac{4}{5} \times \frac{1}{2} = \frac{4 \times 1}{5 \times 2} = \frac{4}{10} = \frac{2}{5}$ 。

计算时可以**先约分再相乘**，避免数字过大。

### 典型例题（Worked Examples）

**例 1.** 计算 $\frac{3}{8} \times \frac{4}{9}$ 。

**解：** 先约分： $3$ 与 $9$ 有公因数 $3$ ， $4$ 与 $8$ 有公因数 $4$ 。

$$\frac{3}{8} \times \frac{4}{9} = \frac{3 \div 3}{8 \div 4} \times \frac{4 \div 4}{9 \div 3} = \frac{1}{2} \times \frac{1}{3} = \frac{1}{6}$$

**例 2.** 一块地有 $\frac{3}{4}$ 公顷，其中 $\frac{2}{3}$ 种了玉米。种玉米的面积是多少公顷？

**解：** 种玉米的面积 $= \frac{3}{4} \times \frac{2}{3} = \frac{3 \times 2}{4 \times 3} = \frac{6}{12} = \frac{1}{2}$ （公顷）。

**例 3.** 计算 $2\frac{1}{3} \times 1\frac{1}{7}$ 。

**解：** 先化成假分数： $2\frac{1}{3} = \frac{7}{3}$ ， $1\frac{1}{7} = \frac{8}{7}$ 。

$$\frac{7}{3} \times \frac{8}{7} = \frac{7 \times 8}{3 \times 7} = \frac{8}{3} = 2\frac{2}{3}$$

### 关键总结（Key Takeaways）

- 分数乘法：分子乘分子，分母乘分母。
- 先约分再乘，计算更简便。
- 带分数相乘时先化成假分数。
- "求一个数的几分之几"用乘法。

### 练一练（Practice）

1. 计算： $\frac{5}{6} \times \frac{3}{10}$ 。
2. 计算： $\frac{7}{12} \times 8$ 。
3. 计算： $1\frac{1}{4} \times 2\frac{2}{5}$ 。
4. 一筐苹果重 $30$ 千克，卖出了 $\frac{2}{5}$ ，卖出了多少千克？

## 分数的除法

### 引入情境（Explore）

一条丝带长 $\frac{3}{4}$ 米，要剪成每段 $\frac{1}{8}$ 米的小段，可以剪多少段？

### 概念建立（Build Understanding）

**分数除以整数**：等于乘以这个整数的倒数。

$$\frac{2}{3} \div 5 = \frac{2}{3} \times \frac{1}{5} = \frac{2}{15}$$

**一个数的倒数**：如果两个数的乘积是 $1$ ，这两个数互为倒数。 $\frac{3}{4}$ 的倒数是 $\frac{4}{3}$ 。

**分数除以分数**：除以一个数等于乘以它的倒数。

$$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} \quad (c \neq 0)$$

例如： $\frac{3}{4} \div \frac{1}{8} = \frac{3}{4} \times \frac{8}{1} = \frac{24}{4} = 6$ 。丝带可以剪成 $6$ 段。

**为什么除以一个分数等于乘以它的倒数？** 因为 $\frac{c}{d} \times \frac{d}{c} = 1$ ，把除法转化为乘法可以利用已知的乘法规则。

### 典型例题（Worked Examples）

**例 1.** 计算 $\frac{5}{6} \div \frac{5}{9}$ 。

**解：**

$$\frac{5}{6} \div \frac{5}{9} = \frac{5}{6} \times \frac{9}{5} = \frac{5 \times 9}{6 \times 5} = \frac{9}{6} = \frac{3}{2} = 1\frac{1}{2}$$

**例 2.** 计算 $4 \div \frac{2}{3}$ 。

**解：**

$$4 \div \frac{2}{3} = 4 \times \frac{3}{2} = \frac{12}{2} = 6$$

**例 3.** 一块长方形土地的面积是 $\frac{4}{5}$ 公顷，宽是 $\frac{2}{5}$ 公顷。长是多少？

**解：** 长 $=$ 面积 $\div$ 宽 $= \frac{4}{5} \div \frac{2}{5} = \frac{4}{5} \times \frac{5}{2} = \frac{20}{10} = 2$ （公顷）。

### 关键总结（Key Takeaways）

- 除以一个数等于乘以它的倒数。
- 倒数：分子分母互换。 $0$ 没有倒数。
- 分数除法的关键步骤：变除为乘，除数取倒数。

### 练一练（Practice）

## 分数应用题

### 引入情境（Explore）

六年级有 $240$ 名学生，其中 $\frac{3}{8}$ 参加了数学兴趣小组。参加数学兴趣小组的有多少人？这类"求一个数的几分之几是多少"的问题，在生活中非常常见。

### 概念建立（Build Understanding）

分数应用题的三种基本类型：

**类型一：求一个数的几分之几是多少**——用乘法。

$$240 \times \frac{3}{8} = 90 \text{（人）}$$

**类型二：已知一个数的几分之几是多少，求这个数**——用除法。

如果参加兴趣小组的 $90$ 人占全年级的 $\frac{3}{8}$ ，求全年级人数： $90 \div \frac{3}{8} = 90 \times \frac{8}{3} = 240$ （人）。

**类型三：求一个数是另一个数的几分之几**——用除法。

$90$ 是 $240$ 的几分之几？ $90 \div 240 = \frac{90}{240} = \frac{3}{8}$ 。

解题关键：找准**单位"1"**（谁是整体）和**对应的分率**（几分之几）。

### 典型例题（Worked Examples）

**例 1.** 一本书有 $300$ 页，小明已经看了 $\frac{2}{5}$ 。他看了多少页？还剩多少页？

**解：**
- 看了的页数： $300 \times \frac{2}{5} = 120$ （页）。
- 剩余页数： $300 - 120 = 180$ （页）。
- 或者直接：剩余分率 $= 1 - \frac{2}{5} = \frac{3}{5}$ ， $300 \times \frac{3}{5} = 180$ （页）。

**例 2.** 水果店运来一批苹果，卖出了 $\frac{3}{4}$ ，正好卖出了 $120$ 千克。这批苹果一共多少千克？

**解：** 单位"1"是这批苹果的总重量。

$$120 \div \frac{3}{4} = 120 \times \frac{4}{3} = 160 \text{（千克）}$$

**例 3.** 甲数是 $48$ ，乙数是 $60$ 。甲数是乙数的几分之几？乙数是甲数的几倍？

**解：**
- 甲是乙的几分之几： $48 \div 60 = \frac{48}{60} = \frac{4}{5}$ 。
- 乙是甲的几倍： $60 \div 48 = \frac{60}{48} = \frac{5}{4} = 1\frac{1}{4}$ （倍）。

**例 4.** 某班有 $45$ 名学生，男生占 $\frac{4}{9}$ 。女生有多少人？

**解：** 女生占全班的 $1 - \frac{4}{9} = \frac{5}{9}$ 。

$$45 \times \frac{5}{9} = 25 \text{（人）}$$

### 关键总结（Key Takeaways）

- "求几分之几是多少"用乘法；"已知几分之几求整体"用除法。
- 解题第一步：找准单位"1"。
- 分率和对应量必须对应同一个单位"1"。
- 画线段图可以帮助理解数量关系。

### 练一练（Practice）

2. 修一条路，已修了 $\frac{2}{7}$ ，正好修了 $800$ 米。这条路全长多少米？

5. 一件衣服打折后便宜了 $36$ 元，便宜的部分占原价的 $\frac{2}{9}$ 。这件衣服原价多少元？

## 参考答案

**练一练 分数的意义**

1. $\frac{3}{8}$ 。
2. $\frac{5}{6}$ 。
3. 平均分成 $10$ 份，取了 $7$ 份。

**练一练 真分数、假分数与带分数**

1. 真分数： $\frac{3}{5}$ 、 $\frac{2}{9}$ 。假分数： $\frac{7}{4}$ 、 $\frac{6}{6}$ 。
2. $\frac{17}{5} = 3\frac{2}{5}$ 。
3. $3\frac{2}{5} = \frac{17}{5}$ 。

**练一练 分数的基本性质**

1. $\frac{15}{45} = \frac{1}{3}$ 。
2. $\frac{28}{42} = \frac{2}{3}$ 。
3. 通分。 $\text{最小公倍数}(6, 9) = 18$ 。 $\frac{5}{6} = \frac{15}{18}$ ， $\frac{7}{9} = \frac{14}{18}$ 。因为 $15 > 14$ ，所以 $\frac{5}{6} > \frac{7}{9}$ 。
4. 公分母为 $\text{最小公倍数}(3, 4, 6) = 12$ 。 $\frac{2}{3} = \frac{8}{12}$ ， $\frac{3}{4} = \frac{9}{12}$ ， $\frac{5}{6} = \frac{10}{12}$ 。

**练一练 分数的加减法**

1. $\frac{2}{3} + \frac{5}{6} = \frac{4}{6} + \frac{5}{6} = \frac{9}{6} = \frac{3}{2} = 1\frac{1}{2}$ 。
2. $\frac{7}{8} - \frac{2}{3} = \frac{21}{24} - \frac{16}{24} = \frac{5}{24}$ 。
3. $4\frac{1}{2} - 1\frac{5}{6} = 4\frac{3}{6} - 1\frac{5}{6}$ 。分数部分不够减，借一： $= 3\frac{9}{6} - 1\frac{5}{6} = 2\frac{4}{6} = 2\frac{2}{3}$ 。

**练一练 分数的乘法**

1. $\frac{5}{6} \times \frac{3}{10} = \frac{5 \times 3}{6 \times 10} = \frac{15}{60} = \frac{1}{4}$ 。
2. $\frac{7}{12} \times 8 = \frac{56}{12} = \frac{14}{3} = 4\frac{2}{3}$ 。
3. $1\frac{1}{4} \times 2\frac{2}{5} = \frac{5}{4} \times \frac{12}{5} = \frac{60}{20} = 3$ 。
4. $30 \times \frac{2}{5} = 12$ （千克）。

**练一练 分数的除法**

1. $\frac{5}{7}$ 的倒数是 $\frac{7}{5}$ ， $3$ 的倒数是 $\frac{1}{3}$ 。
2. $\frac{7}{8} \div \frac{3}{4} = \frac{7}{8} \times \frac{4}{3} = \frac{28}{24} = \frac{7}{6} = 1\frac{1}{6}$ 。
3. $2\frac{1}{2} \div 1\frac{1}{4} = \frac{5}{2} \div \frac{5}{4} = \frac{5}{2} \times \frac{4}{5} = \frac{20}{10} = 2$ 。
4. $\frac{5}{6} \div \frac{1}{6} = \frac{5}{6} \times 6 = 5$ （段）。

**练一练 分数应用题**

1. $2400 \times \frac{1}{6} = 400$ （本）。
2. $800 \div \frac{2}{7} = 800 \times \frac{7}{2} = 2800$ （米）。
3. $450 \div 600 = \frac{450}{600} = \frac{3}{4}$ 。
4. 梨树： $120 \times \frac{3}{4} = 90$ （棵）。两种树共 $120 + 90 = 210$ （棵）。
5. $36 \div \frac{2}{9} = 36 \times \frac{9}{2} = 162$ （元）。
