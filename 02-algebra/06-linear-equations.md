## 方程的概念与方程的解

### 引入情境（Explore）

小明去文具店买了 $3$ 支相同的钢笔和一个 $8$ 元的笔袋，一共花了 $35$ 元。每支钢笔多少钱？我们可以设每支钢笔 $x$ 元，列出等式 $3x + 8 = 35$ 。这个含有未知数的等式就是一个**方程**，而使这个等式成立的 $x$ 的值就是方程的**解**。

### 概念建立（Build Understanding）

**方程**：含有未知数的等式叫做方程。

**方程的解**：使方程左右两边相等的未知数的值，叫做方程的解。

**一元一次方程**：只含有**一个未知数**（一元），未知数的最高次数为 ** $1$ **（一次）的方程。一般形式为：

$$
ax + b = 0 \quad (a \neq 0)
$$

**为什么 $a \neq 0$ ？** 如果 $a = 0$ ，方程变成 $b = 0$ ，不含未知数，就不是一元一次方程了。

### 典型例题（Worked Examples）

**例 1.** 判断下列方程中，哪些是一元一次方程？

(1) $2x + 5 = 11$ 　(2) $x^2 - 3 = 0$ 　(3) $x + y = 7$ 　(4) $\frac{x}{2} = 4$

**解：**

(1) 一个未知数，最高次数为 $1$ 。✓ 是一元一次方程。

(2) 未知数的最高次数为 $2$ 。✗ 是一元二次方程。

(3) 有两个未知数。✗ 是二元一次方程。

(4) $\frac{x}{2} = 4$ 即 $\frac{1}{2}x = 4$ ，一个未知数，最高次数为 $1$ 。✓ 是一元一次方程。

**例 2.** 检验 $x = 9$ 是否为方程 $3x + 8 = 35$ 的解。

**解：** 将 $x = 9$ 代入左边： $3 \times 9 + 8 = 27 + 8 = 35$ 。左边 $=$ 右边。所以 $x = 9$ 是方程的解。

**例 3.** 检验 $x = 2$ 是否为方程 $5x - 3 = 8$ 的解。

**解：** 代入左边： $5 \times 2 - 3 = 10 - 3 = 7 \neq 8$ 。左边 $\neq$ 右边，所以 $x = 2$ 不是方程的解。

### 关键总结（Key Takeaways）

- 方程是含有未知数的等式。
- 一元一次方程：一个未知数，最高次数为 $1$ 。
- 检验方程的解：将值代入看等式是否成立。

### 练一练（Practice）

## 等式的基本性质

### 引入情境（Explore）

想象一架天平，左右两边放的东西一样重。如果在两边同时加上相同的重量，天平仍然平衡；如果两边同时取走相同的重量，天平仍然平衡。这就是等式的基本性质——也是解方程的理论基础。

### 概念建立（Build Understanding）

**等式的基本性质**：

**性质 1**：等式两边同时加上（或减去）同一个数或式子，等式仍然成立。

$$
\text{若 } a = b, \text{ 则 } a + c = b + c, \quad a - c = b - c
$$

**性质 2**：等式两边同时乘以（或除以）同一个不为零的数，等式仍然成立。

$$
\text{若 } a = b, \text{ 则 } ac = bc, \quad \frac{a}{c} = \frac{b}{c} \; (c \neq 0)
$$

**为什么这些性质重要？** 解方程的每一步操作，本质上都是在使用等式的基本性质，逐步将方程化简为 $x = \text{某个值}$ 的形式。

### 典型例题（Worked Examples）

**例 1.** 利用等式性质解方程 $x + 7 = 15$ 。

**解：** 两边同时减去 $7$ （性质 1）。

$$x + 7 - 7 = 15 - 7$$

$$x = 8$$

**例 2.** 利用等式性质解方程 $3x = 18$ 。

**解：** 两边同时除以 $3$ （性质 2）。

$$\frac{3x}{3} = \frac{18}{3}$$

$$x = 6$$

**例 3.** 利用等式性质解方程 $\frac{x}{4} - 3 = 2$ 。

**解：**

- 两边加 $3$ ： $\frac{x}{4} = 5$ 。
- 两边乘以 $4$ ： $x = 20$ 。

### 关键总结（Key Takeaways）

- 等式两边做相同的加减乘除运算（除数不为零），等式不变。
- 等式的性质是解方程的理论依据。
- 解方程就是利用等式性质逐步"隔离"未知数。

### 练一练（Practice）

2. 利用等式性质解 $-4x = 20$ 。

4. 利用等式性质解 $2x + 1 = 9$ 。
5. 为什么等式两边不能同时除以 $0$ ？

## 解一元一次方程

### 引入情境（Explore）

在上一节中，我们用等式的基本性质一步一步地求解简单方程。对于更复杂的方程，如 $3(2x - 1) = 4x + 5$ ，我们需要系统化的解题步骤：去括号、移项、合并同类项、系数化为 $1$ 。

### 概念建立（Build Understanding）

解一元一次方程的**标准步骤**：

1. **去分母**：方程中有分数时，两边乘以各分母的最小公倍数。
2. **去括号**：利用分配律展开括号。
3. **移项**：把含未知数的项移到等号一边，常数项移到另一边。移项要**变号**。
4. **合并同类项**：将同类项合并。
5. **系数化为 $1$ **：两边除以未知数的系数。

**移项**的本质是等式性质 1 的简化写法：将一项从等号一边"搬"到另一边，符号取反。

### 典型例题（Worked Examples）

**例 1.** 解方程 $5x - 3 = 2x + 9$ 。

**解：**

移项（含 $x$ 的项移到左边，常数项移到右边）：

$$5x - 2x = 9 + 3$$

合并同类项：

$$3x = 12$$

系数化为 $1$ ：

$$x = 4$$

**例 2.** 解方程 $3(2x - 1) = 4x + 5$ 。

**解：**

去括号： $6x - 3 = 4x + 5$ 。

移项： $6x - 4x = 5 + 3$ 。

合并： $2x = 8$ 。

系数化为 $1$ ： $x = 4$ 。

**例 3.** 解方程 $\frac{x - 1}{2} + 1 = \frac{2x + 1}{3}$ 。

**解：**

去分母（两边乘以 $6$ ）： $3(x - 1) + 6 = 2(2x + 1)$ 。

去括号： $3x - 3 + 6 = 4x + 2$ 。

化简： $3x + 3 = 4x + 2$ 。

移项： $3x - 4x = 2 - 3$ 。

合并： $-x = -1$ 。

系数化为 $1$ ： $x = 1$ 。

**例 4.** 解方程 $0.3x + 1 = 0.5x - 0.4$ 。

**解：** 先将小数化为整数（两边乘以 $10$ ）： $3x + 10 = 5x - 4$ 。

移项： $3x - 5x = -4 - 10$ 。

合并： $-2x = -14$ 。

$x = 7$ 。

### 关键总结（Key Takeaways）

- 解方程的标准流程：去分母 → 去括号 → 移项 → 合并 → 系数化 $1$ 。
- 移项一定要变号，这是最常见的错误来源。
- 有小数时先化为整数，有分数时先去分母。

### 练一练（Practice）

1. 解方程 $4x - 7 = x + 8$ 。

3. 解方程 $\frac{x}{3} - \frac{x - 1}{2} = 1$ 。

## 一元一次方程的应用

### 引入情境（Explore）

数学源于生活，也要服务于生活。一元一次方程可以解决许多实际问题——行程问题、工程问题、浓度问题等。关键是学会**审题列方程**：找出等量关系，用方程语言表达出来。

### 概念建立（Build Understanding）

列方程解应用题的步骤：

1. **审题**：理解题意，找出已知量和未知量。
2. **设未知数**：用 $x$ 表示所求的量。
3. **找等量关系**：从题目中找到一个等量关系。
4. **列方程**：用代数式表达等量关系。
5. **解方程**：求出 $x$ 的值。
6. **检验并作答**：检验解是否符合题意，写出完整答案。

### 典型例题（Worked Examples）

**例 1.**（行程问题）甲、乙两人从相距 $36$ 千米的两地同时出发，相向而行。甲的速度为 $4$ 千米/时，乙的速度为 $5$ 千米/时。多少小时后两人相遇？

**解：** 设 $x$ 小时后相遇。

等量关系：甲走的路程 $+$ 乙走的路程 $=$ 总路程。

$$4x + 5x = 36$$

$$9x = 36$$

$$x = 4$$

答： $4$ 小时后两人相遇。

**例 2.**（工程问题）一项工程，甲单独完成需 $12$ 天，乙单独完成需 $18$ 天。两人合做多少天完成？

**解：** 设合做 $x$ 天完成。

甲每天完成 $\frac{1}{12}$ ，乙每天完成 $\frac{1}{18}$ 。 $x$ 天合做完成量 $= 1$ 。

$$\frac{x}{12} + \frac{x}{18} = 1$$

去分母（乘以 $36$ ）： $3x + 2x = 36$ ， $5x = 36$ ， $x = 7.2$ 。

答：合做 $7.2$ 天完成（即 $7$ 天零 $4.8$ 小时）。

**例 3.**（浓度问题）有 $500$ 克含盐 $10\%$ 的盐水，要使盐水浓度变为 $20\%$ ，需要再加多少克盐？

**解：** 设需加 $x$ 克盐。

原有盐量： $500 \times 10\% = 50$ 克。

加盐后盐的总量为 $(50 + x)$ 克，溶液总量为 $(500 + x)$ 克。

$$\frac{50 + x}{500 + x} = 20\%$$

$$50 + x = 0.2(500 + x)$$

$$50 + x = 100 + 0.2x$$

$$0.8x = 50$$

$$x = 62.5$$

答：需再加 $62.5$ 克盐。

**例 4.**（年龄问题）父亲今年 $45$ 岁，儿子今年 $15$ 岁。多少年前，父亲的年龄是儿子的 $5$ 倍？

**解：** 设 $x$ 年前。

$$45 - x = 5(15 - x)$$

$$45 - x = 75 - 5x$$

$$4x = 30$$

$$x = 7.5$$

答： $7.5$ 年前，父亲年龄是儿子的 $5$ 倍。

### 关键总结（Key Takeaways）

- 列方程的关键是找到题目中的等量关系。
- 行程问题的基本关系：路程 $=$ 速度 $\times$ 时间。
- 工程问题的基本关系：工作量 $=$ 效率 $\times$ 时间。
- 浓度问题的基本关系：溶质 $=$ 溶液 $\times$ 浓度。

### 练一练（Practice）

1. 一个数的 $3$ 倍比它本身大 $14$ ，求这个数。
2. 甲、乙两人从同一地点同方向出发，甲的速度为 $6$ 千米/时，乙的速度为 $4$ 千米/时。甲先出发 $1$ 小时，乙出发多少小时后追上甲？
3. 一批零件由甲单独加工需要 $10$ 天，由乙单独加工需要 $15$ 天。甲先做了 $4$ 天后，乙接着做，还需要多少天完成？
4. 一件商品打八折后售价为 $240$ 元，原价是多少？
5. 小明有 $50$ 元钱去买练习本，每本 $3$ 元。他最多能买多少本？还剩多少钱？
6. 有 $200$ 克含糖 $15\%$ 的糖水，要使浓度变为 $10\%$ ，需加多少克水？
