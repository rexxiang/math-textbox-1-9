# 模块二：代数

> **适用年级**：7-9 年级

## 模块导言

代数是数学从"具体"走向"抽象"的关键一步。在模块一"数与运算"中，我们学习了整数、分数、小数等**具体的数**以及它们的运算法则。进入代数，我们将用**字母代替数**，把具体的计算规律提升为通用的表达式和公式——这一跨越，使数学获得了描述一般规律、解决未知问题的强大能力。

代数模块的主线可以概括为三条发展脉络：

1. **数的扩展**：有理数（§2.1）→ 实数（§2.2），数系从整数、分数扩展到包含负数和无理数。
2. **式与方程**：代数式（§2.3）→ 因式分解（§2.4）→ 分式（§2.5）→ 一元一次方程（§2.6）→ 二元一次方程组（§2.7）→ 一元二次方程（§2.8）→ 不等式（§2.9），从"写出关系"到"求出未知"。
3. **函数与图象**：一次函数（§2.10）→ 二次函数（§2.11），从静态等式走向动态变化的研究。

## 与模块一的衔接

代数模块与"数与运算"模块紧密相连：

- §2.1 有理数 直接建立在 §1.1 整数认识、§1.5 分数、§1.6 小数的基础上，将数系扩展到负数。
- §2.3 代数式 需要运用 §1.3 运算律（交换律、结合律、分配律）进行整式运算。
- §2.5 分式 的运算规则类比 §1.5 分数的通分与约分。

学好模块一的运算基础，是顺利进入代数学习的前提。

## 学习建议

1. **按顺序推进**：本模块各章的前置关系层层递进，建议按 §2.1 → §2.11 的顺序学习。
2. **重视运算**：代数的核心技能是运算，每个章节的练习要认真完成。
3. **画图辅助**：数轴、函数图象是理解代数概念的重要工具，养成画图的习惯。
4. **联系实际**：每章的引入情境展示了代数在生活中的应用，帮助你理解"为什么要学这些"。
