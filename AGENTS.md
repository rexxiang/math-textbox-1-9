# AGENTS.md

## 项目概述

本仓库包含一本覆盖中国小学一年级到初中三年级（Grade 1–9）的完整数学自学教材。教材按照美国 Common Core 标准的教学风格编排，以"探索 → 建立概念 → 例题 → 总结 → 练习"为核心教学结构。

## 仓库结构

```
├── AGENTS.md              # 本文件：项目说明与协作指南
├── 00-导读与目录.md         # 全书导读、学习路径、符号约定
├── 01-数与运算.md           # 主线一：整数→四则运算→分数→小数→百分数→比与比例
├── 02-代数.md              # 主线二：有理数→实数→代数式→方程→不等式→函数
├── 03-几何.md              # 主线三：点线面→三角形→四边形→圆→坐标几何→三角函数
├── 04-统计与概率.md         # 主线四：图表→集中趋势→离散程度→概率→散点图
├── .github/
│   └── workflows/
│       └── build-pdf.yml  # CI：Markdown → PDF 转换与 Release 发布
└── .gitignore
```

## 内容编写规范

### 教学结构

每个知识点必须包含以下五个部分（使用 `####` 标题）：

1. **引入情境（Explore）**：以具体问题或生活场景引入
2. **概念建立（Build Understanding）**：从直觉到严格定义
3. **典型例题（Worked Examples）**：1–2 道带分步解析的例题
4. **关键总结（Key Takeaways）**：3–5 句核心要点
5. **练一练（Practice）**：2–3 道自测题，答案用 `<details>` 折叠

### 模块级总结

每章结束时包含：
- **易错提醒**（`> ⚠️ **易错提醒**：...`）
- **知识拓展**（`> 💡 **知识拓展**：...`）
- **知识脉络回顾**

### LaTeX 公式规范

- 行内公式：`$...$`；独立公式：`$$...$$`
- 使用 `\geq` / `\leq`（不用 `\geqslant` / `\leqslant`）
- 绝对值：`\lvert a \rvert`
- 整除：`a \mid b`
- 多行对齐：`aligned` 环境（不用 `align`）
- 中文文本在公式中用 `\text{...}` 包裹

### 图形

- 优先使用内嵌 SVG
- 每个 SVG 附带 `<!-- 文字描述：[说明] -->` 注释
- SVG 宽度不超过 500px

### 交叉引用

使用 `§` + 章节编号（如"参见 §4.2"）

## 统计与概率范围限定

严格遵循 Common Core 标准，**不得包含**以下高中内容：
方差、标准差、正态分布、条件概率、独立性检验、排列组合公式、相关系数、回归方程、随机变量、期望值。

## CI/CD

- 推送 tag（`v*`）时自动触发 PDF 构建
- 使用 Pandoc + XeLaTeX 将 Markdown 转换为 PDF
- 生成的 PDF 通过 GitHub Release 发布
