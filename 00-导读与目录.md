# 数学自学教材：小学一年级到初中三年级

> 本教材覆盖中国小学一年级至初中三年级的全部核心数学内容，按照美国 Common Core 标准的教学风格编排，以"探索 → 建立概念 → 例题 → 总结 → 练习"为核心教学结构。无论你是查漏补缺还是系统自学，都可以在这里找到适合自己的学习路径。

---

## 一、如何使用本教材

### 1.1 学习路径建议

本教材的内容大致对应两个学段，建议根据自身情况选择不同的学习模式：

**小学阶段（Grade 1–5）内容 → 复习查漏模式**

- 涵盖整数、四则运算、运算律、因数与倍数、分数、小数等基础内容。
- 如果你已经完成小学学业，建议以**快速浏览 + 重点突破**的方式使用：通读"关键总结"部分，遇到不熟悉的知识点再回头细读"概念建立"与"典型例题"。
- 重点关注每章末尾的"易错提醒"，这些是历年考试中出错率最高的地方。

**初中阶段（Grade 6–9）内容 → 系统学习模式**

- 涵盖有理数、实数、代数式、方程、不等式、函数、几何证明、概率统计等核心内容。
- 建议按照章节顺序，完整经历每个知识点的五步教学流程：
  1. 先独立思考"引入情境"中的问题；
  2. 再阅读"概念建立"，将直觉认识上升为严格定义；
  3. 对照"典型例题"检验自己的理解；
  4. 用"关键总结"巩固核心要点；
  5. 最后通过"练一练"自测掌握程度。

### 1.2 阅读建议

- **不要跳过引入情境**：每个知识点的引入情境都经过精心设计，帮助你理解"为什么需要这个概念"。
- **动笔计算**：数学不是"看懂就会"的学科，请务必在纸上亲手完成每一道例题和练习。
- **善用交叉引用**：文中以 `§` 标记的引用（如"参见 §1.3"）指向相关知识点，帮助你建立知识之间的联系。
- **反复回顾知识脉络**：每章末尾的"知识脉络回顾"帮助你把握全局，建议定期重读。

## 二、符号约定

本教材使用 LaTeX 语法书写数学公式，以下是常见符号的说明：

| 符号 | LaTeX 写法 | 含义 |
|------|-----------|------|
| $a + b$ | `$a + b$` | 行内公式，嵌在正文中 |
| $$a^2 + b^2 = c^2$$ | `$$a^2 + b^2 = c^2$$` | 独立公式，单独占一行 |
| $\geq$, $\leq$ | `\geq`, `\leq` | 大于等于、小于等于 |
| $\lvert a \rvert$ | `\lvert a \rvert` | 绝对值 |
| $a \mid b$ | `a \mid b` | $a$ 整除 $b$ |
| $\triangle ABC$ | `\triangle ABC` | 三角形 $ABC$ |
| $\angle A$ | `\angle A` | 角 $A$ |
| $\overrightarrow{AB}$ | `\overrightarrow{AB}` | 从 $A$ 到 $B$ 的射线或向量 |
| $\parallel$, $\perp$ | `\parallel`, `\perp` | 平行、垂直 |

### 交叉引用格式

- **章内引用**：`§2.3` 表示本章第 2.3 节。
- **跨章引用**：`§1-3.2` 表示第 01 章（数与运算）的第 3.2 节。
- 文件对应关系：`§1` = `01-数与运算.md`，`§2` = `02-代数.md`，`§3` = `03-几何.md`，`§4` = `04-统计与概率.md`。

### 特殊标记

- `> ⚠️ **易错提醒**：...` — 标记常见错误与易混淆点。
- `> 💡 **知识拓展**：...` — 标记超出主线但有助于深化理解的内容。
- `<details>` 折叠块 — 用于练习题答案，请先独立完成再查看。

## 四、知识脉络总览

贯穿四章内容的，有两条不易察觉却至关重要的暗线。理解这两条线索，能帮助你把握全书的内在逻辑。

### 4.1 暗线一：数系的扩充

数学的发展史，在很大程度上是"数"不断扩充的历史。本教材中，数系的扩充路径如下：

$$
\text{自然数} \;\xrightarrow{\text{引入 } 0 \text{ 和负数}}\; \text{整数} \;\xrightarrow{\text{引入分数}}\; \text{有理数} \;\xrightarrow{\text{引入无理数}}\; \text{实数}
$$

每一次扩充都源于**实际需要**：

- **自然数 → 整数**（§1.1 → §2.1）：为了表示"不足"和"相反方向"，引入 $0$ 和负数。
- **整数 → 有理数**（§2.1 → §1.5）：为了精确表示"部分"和"比率"，引入分数（即有理数的稠密化）。
- **有理数 → 实数**（§2.1 → §2.2）：为了求解 $x^2 = 2$ 这样的方程，引入无理数，填满数轴上的每一个"空隙"。

> 💡 **知识拓展**：每次数系扩充后，原有的运算律（交换律、结合律、分配律）仍然成立——这并非偶然，而是数学家在定义新运算时刻意保持的"一致性"。

### 4.2 暗线二：工具的升级

解决问题的工具也在不断升级，从具体的算术计算到抽象的函数关系：

$$
\text{算术} \;\xrightarrow{\text{用字母代替数}}\; \text{代数式} \;\xrightarrow{\text{建立等量关系}}\; \text{方程} \;\xrightarrow{\text{研究变化规律}}\; \text{函数}
$$

- **算术 → 代数式**（§1 → §2.3）：用字母表示未知数或一般化的数量关系，将具体问题抽象为一般规律。
- **代数式 → 方程**（§2.3 → §2.6–§2.8）：当代数式中包含等号和未知数时，就构成方程——一个"寻找未知数"的工具。
- **方程 → 函数**（§2.6 → §2.10–§2.11）：当关注的不再是单个未知数的值，而是两个变量之间的对应关系时，函数应运而生。

> 💡 **知识拓展**：函数的本质是"对应关系"。一次函数 $y = kx + b$ 描述的是匀速变化，二次函数 $y = ax^2 + bx + c$ 描述的是加速变化——自然界中的许多现象都可以用这两种模型来近似。

## 五、四条知识主线的交汇

数与运算、代数、几何、统计与概率并非孤立的四个板块，它们在许多地方彼此交汇、相互支撑：

### 5.1 数与运算 ↔ 代数

- 算术中的四则运算（§1.2–§1.3）是代数运算（§2.3）的特殊情形：当字母取具体数值时，代数式就退化为算术表达式。
- 分数的运算（§1.5）直接延伸为分式的运算（§2.5），方法完全一致，只是将数字换成了含字母的表达式。
- 比与比例（§1.8）是一次函数（§2.10）中"正比例关系"的算术原型。

### 5.2 代数 ↔ 几何

- 勾股定理（§3.7）的代数表达 $a^2 + b^2 = c^2$ 将几何图形与代数方程联系在一起。
- 坐标几何（§3.9）是代数与几何最深刻的交汇点：用代数方程描述几何图形，用几何图像理解代数关系。
- 一次函数的图像（§2.10）是一条直线，二次函数的图像（§2.11）是一条抛物线——函数的性质可以通过图形直观理解。

### 5.3 几何 ↔ 统计与概率

- 扇形图（§4.1）依赖于圆与扇形的几何知识（§3.6）。
- 概率中的"几何概型"思想，虽然不在本教材的正式范围内，但散点图（§4.6）中对区域的直观划分已经蕴含了几何与概率结合的萌芽。

### 5.4 代数 ↔ 统计与概率

- 加权平均数（§4.2）的计算本质上是代数式的求值。
- 散点图中判断正相关与负相关（§4.6），需要借助一次函数（§2.10）"斜率"的直觉。
- 概率的计算（§4.5）经常需要分数与百分数的运算能力（§1.5、§1.7）。

## 六、学习进度参考

以下是建议的学习顺序，箭头表示前置依赖关系：

```
第一章（§1.1–§1.8）
    ├──→ 第二章前半（§2.1–§2.5）
    │       └──→ 第二章后半（§2.6–§2.11）
    │               └──→ 第三章后半（§3.7–§3.10）
    └──→ 第三章前半（§3.1–§3.6）
    └──→ 第四章（§4.1–§4.6）
```

- 第一章是所有后续内容的基础，应最先完成。
- 第三章前半（基础几何）与第二章可以并行学习。
- 第四章（统计与概率）相对独立，可在任何阶段穿插学习，但§4.6 需要一次函数的基础。
- 第三章后半（坐标几何、三角函数）依赖第二章的函数知识，应放在最后。
