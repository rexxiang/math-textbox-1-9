# 质量打磨第 2 轮

## 本轮目标

削掉 `§6.2` 的认知落差，让学生第一次尝试的题型更快获得正反馈。

## 本轮发现

### `§6.2` 的首个 payoff 来得太晚

- tryit 让学生先做的是最常见的 `x^2 + 5x + 6`。
- 但 blueprint 要一路经过平方差、`x^4 - 1`、完全平方，才回到这一型。
- 这会让本节显得比真实核心更难，也削弱了“先试一试 → 立刻看见同型样板”的体验。

## 本轮修复

- `06-algebra-deepening/02-factorization.typ`
  - 把 `x^2 + b x + c` 这一基本型前移成 blueprint 里的第一批核心例题。
  - 调整例号与 tryit 的对应关系，让学生做完 tryit 后能直接找到“同题型的最近样板”。
  - 把更绕的 `x^4 - 1` 留在后面，不再抢占入口位置。

## 本轮涉及文件

- `typst/06-algebra-deepening/02-factorization.typ`
