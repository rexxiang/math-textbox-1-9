# 质量打磨第 3 轮

## 本轮目标

把几何测量链里最容易“背结论、忘发动机”的两节重新排顺：

1. `§7.4` 要更像一条证明链，而不是结论表。
2. `§7.6` 要先做出一次测量，再去记特殊角和术语。

## 本轮发现

### 1. `§7.4` 的证明发动机还不够亮

- 本节危机与发现都在说“从半径一路推出稳定规律”。
- 但 blueprint 容易被读成“术语 + 定理 + 性质”的列表，学生可能只记住结果。

### 2. `§7.6` 还在入口处堆了太多项目

- 定义、特殊角、路线表、仰角俯角、坡度、步骤清单都集中在前半段。
- 对第一次接触三角比的读者，这比一次真实测量更抽象，也更重。

## 本轮修复

- `07-geometry-deepening/04-circles.typ`
  - 在 blueprint 一开始直接亮出“连半径 → 等腰 / 全等 → 弦角结论”的核心引擎。
  - 让学生先看到圆题为什么会退回三角形题，再去记结论。
- `07-geometry-deepening/06-trigonometric-measurement.typ`
  - 重排 blueprint 顺序为“定义 → 一次真实测量 → 特殊角 → 仰角俯角坡度 → 选比值 checklist”。
  - 合并原来的路线表与步骤表，减轻入口负担。

## 本轮涉及文件

- `typst/07-geometry-deepening/04-circles.typ`
- `typst/07-geometry-deepening/06-trigonometric-measurement.typ`
