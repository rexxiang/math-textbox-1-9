# SSOT 版本说明

- **canonical（机器可读）**：`docs/knowledge-graph/book-graph.yaml`
- **human-view（人类阅读）**：`docs/knowledge-graph/tool-dependencies.md`
- **Typst label canonical**：`<tool:...>` / `<ch:...>` / `<meta:...>`。工具 label 的 slug 与 `book-graph.yaml` 的 node-id 一致，例如 node-id `pf01-decimal-system` 对应 `<tool:pf01-decimal-system>`。
- **`#secref` 兼容机制**：`#secref("tool:pf01-decimal-system")` 是显式 canonical；`#secref("pf01-decimal-system")` 是兼容简写，`theme-v2.typ` 与 `scripts/validate_secrefs.py` 都会自动补成 `tool:`。`ch:` / `meta:` 必须显式写前缀。
- **工具 slug 前缀**（详见 `tool-dependencies.md` 头部表格）：
  | 前缀 | 含义 | 源码目录 |
  |---|---|---|
  | pf | public foundation | `typst/1-public-foundation/` |
  | cb | common bridges | `typst/2-common-bridges/` |
  | al | algebra & symbols | `typst/3a-algebra-symbols/` |
  | ge | geometry & proof | `typst/3b-geometry-proof/` |
  | fn | functions & change | `typst/3c-functions-change/` |
  | dt | data & uncertainty | `typst/3d-data-uncertainty/` |
- **book-graph node 类型**：
  - `knowledge-point`：进入主线学习 DAG 的知识点工具，允许出现在 `prerequisites` / `dependencies`。
  - `chapter-meta`：章首页、总检、总结、readiness contract 等元节点；只表达结构与复习导航，不作为新知识点依赖。
  - `optional-bridge`：`☞ 高中桥` / `☞ 继续往前` 等可选延伸节点；不得进入任何主线 `prerequisites`，也不作为后续章节的必备前置。
- **校验入口**：
  - `make kg-validate-book` → `scripts/validate_book_graph.py docs/knowledge-graph/book-graph.yaml`
  - `make validate-secrefs` → `scripts/validate_secrefs.py`
- **命名约定**：历史命名空间 `kp:` 已废弃；新 label 使用 `tool:` / `ch:` / `meta:`。不要把 node-id 写成 `<pf01-...>` 这类无命名空间 Typst label。
- **prerequisites 范围**：`book-graph.yaml` 的 `prerequisites` 只记录主线必备依赖，不包含 `☞ 高中桥` / `☞ 继续往前` 或挑战题、选做题中出现的延伸工具。
- **上次同步**：2026-04-21

## anchors_defined.txt / secref_called.txt 的生成

这两个文件**不被任何校验脚本读取**，仅作为人类查阅的快照。`anchors_defined.txt` 保存 canonical label（带 `tool:` / `ch:` / `meta:`）；`secref_called.txt` 保存源码中的原始调用，所以可能同时出现 `pf01-...` 与 `tool:pf01-...`。这与 `validate_secrefs.py` 的解析规则一致。重生成命令（在仓库根执行）：

```bash
python3 -c "
import re
from pathlib import Path
ANCHOR_RE = re.compile(r'<(tool|ch|meta):([^>]+)>')
SECREF_RE = re.compile(r'#secref\(\"([^\"]+)\"\)')
anchors, secrefs = set(), set()
for p in Path('typst').rglob('*.typ'):
    txt = p.read_text(encoding='utf-8')
    for kind, slug in ANCHOR_RE.findall(txt):
        anchors.add(f'{kind}:{slug}')
    for slug in SECREF_RE.findall(txt):
        secrefs.add(slug)
header = '# auto-generated by scripts oneliner — do not edit. Regenerate with the snippet in docs/SSOT-VERSION.md.\n'
Path('anchors_defined.txt').write_text(header + '\n'.join(sorted(anchors)) + '\n', encoding='utf-8')
Path('secref_called.txt').write_text(header + '\n'.join(sorted(secrefs)) + '\n', encoding='utf-8')
"
```
